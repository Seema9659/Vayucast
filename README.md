🌦️ VayuCast – Weather Forecast Web App

VayuCast is a modern, responsive weather forecasting web application that provides real-time weather information for cities around the world. Built using HTML, CSS, and JavaScript, it integrates with the OpenWeatherMap API and Open-Meteo API to deliver accurate weather data in a clean and interactive interface.

✨ Features
🔍 Search weather by city name
🌡️ Real-time temperature and "Feels Like" information
💧 Humidity, wind speed, and atmospheric pressure
🌅 Sunrise and sunset timings
☀️ UV Index
🌬️ Air Quality Index (AQI)
👀 Visibility information
🌧️ Rain probability
📅 5-Day Weather Forecast
🌤️ Dynamic weather animations (sun, clouds, rain, snow, fog, thunderstorms)
💾 Remembers the last searched city using Local Storage
📱 Responsive glassmorphism-inspired UI
🛠️ Technologies Used
HTML5
CSS3
JavaScript (ES6)
OpenWeatherMap API
Open-Meteo API

🚀 Getting Started
Clone the repository.
Open index.html in your browser.
Replace the API key in script.js with your own OpenWeatherMap API key (if needed).
Start searching for any city to view its weather.
